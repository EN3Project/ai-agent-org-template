# Runtime OBSERVATIONS

このファイルは、運用中の摩擦や改善サインを軽量に残すためのログである。
初期状態では空に近いまま使う。作業本文や詳細ログを保存する場所ではない。

## Rules

- 機密情報、個人情報、顧客情報、認証情報を残さない。
- ユーザー入力や作業本文をそのまま残さない。
- 構造的な観測だけを書く。
- 改善提案はユーザー承認なしに実行しない。
- 細かい作業ログは必要時だけ `Scratch/` または `Reports/` に分ける。
